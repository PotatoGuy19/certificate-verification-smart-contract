
Please follow these instructions to start the web app properly.

1. Download Node.js at "https://nodejs.org/en/"

2. Open terminal and navigate to this directory.

3. Enter "npm" to reinstall all the dependencies.

4. Enter "npm install -g ethereumjs-testrpc" to install TestRPC

5. Enter "npm i -S express" to install Express.js

6. Enter "npm i -S web3" to install web3.js

7. Enter "testrpc" to start the Ethereum test-net

8. Open a SEPARATE TAB and enter the command "npm start"

9. Open browser and go to "localost: 3000"
